
# IN1166-XAI

## Descrição

Projeto acadêmico focado em Explainable Artificial Intelligence (XAI) - Inteligência Artificial Explicável.

## Objetivo

Implementar técnicas de explicabilidade em modelos de inteligência artificial no domínio automotivo.


## Requisitos

- Python 3.10
- pip ou conda

## Instalação

```bash
pip install -r requirements.txt
```

## Como Usar

Executar 

```bash
python ./intrusion_detection_ch.py
```

ou 

```bash
python ./intrusion_detection_main.py
```
